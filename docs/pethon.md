# ¿Cómo es que funciona Pethon?
Pethon es un intérprete simple que hace una tarea, y esa tarea es convertir código fuente
escrito con *palabras clave* en español, a su equivalente en Python.

Un ejemplo de esto es el siguiente código :
```
edad = 16

si edad >= 18:
    imprimir("Mayor de edad")
sino:
    imprimir("Menor de edad")
```
Que Pethon transformará en :
```
edad = 16

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")
```

Después de hacer la conversión, Pethon se encarga de ejecutar el programa
